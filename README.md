# Toll_Collection_System

## Overview

The Toll Collection System is a Python-based project designed to manage toll booth operations efficiently. This system helps in vehicle entry management, toll fee calculation, data storage, and record handling using Python file handling techniques.

## Features

* Vehicle entry and exit management
* Toll fee calculation
* Store toll records in files
* Read and update vehicle data
* File handling for data management
* Error and exception handling

## Technologies Used

* Python
* File Handling

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/Toll_Collection_System.git
```

2. Open the project folder:

```bash
cd Toll_Collection_System
```

3. Run the project:

```bash
python main.py
```
